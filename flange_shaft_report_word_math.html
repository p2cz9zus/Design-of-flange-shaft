Technical Report

工作機械の回転軸用
フランジシャフト

駆動モーターからの動力を仲介する中間軸用（加工には直接関係ない部分用）の設計・開発，強度計算検証，およびJIS規格に基づく選定基準を記録したものである．

設計理念

01 / MINIMAL PARTS

部品点数をなるべく抑える

構造をシンプルに保ち，故障リスクおよび組立の複雑性を排除するための最適化アプローチ．

02 / BALANCE

強度とコストが両立できる設計

過剰設計を避けつつ，実用回転に耐えうる最適なマテリアルと寸法のバランスを定義．

03 / PRECISION

組立後，フランジとシャフトが直角になっている必要あり

軸ブレや振動を極小化するため，幾何学的な直角度の確保を絶対的な設計要件として設定．

04 / OPTIMIZATION

丸材と棒材の「キー溝」を採用

一体削り出しだと材料の79%を捨てることになり，コスト高になるため，セパレート構造とキー溝による締結メカニズムを確立．

部品仕様と加工プロセス

部品図（フランジ）

FLANGE SPECIFICATION

【フランジの加工】

  • フランジの段差部分はエンドミルで加工する．
  • フランジの穴加工 → 初めにエンドミルで加工し，リーマー加工で表面仕上げを行う．
  • フィレット部分 → ワイヤ放電加工

部品図（シャフト）

SHAFT SPECIFICATION

【シャフトの加工】

  • 丸棒を切削し，シャフトを掘り出す．
  • キー溝 → 小型エンドミルを用いた，ミーリング加工

部品図（キー）

KEY SPECIFICATION

【キーの加工】

  • 角材の切削及びワイヤ放電加工

組立の流れ

  • 丸材（フランジ）へ（シャフト）を挿入 ＋ キーで固定

【課題と解決策】

  • フランジにキー溝を設置し，必要なトルクに耐えられるようにする
    → 応力が集中しやすいキー溝の形をどうするか？ → フィレットをつける
  • 材料の強度確保
    → 調質（焼き入れ・焼き戻し）を行い，強度と靭性を向上させる．
  • 強度計算による信頼性の担保

フランジシャフトの回転メカニズム

STEP 01

① フランジが回る

STEP 02

② キーの上側 (4mm) が押される

STEP 03

③ キー全体が回ろうとする

STEP 04

④ キーの下側 (6mm) がシャフトを押す

STEP 05

⑤ シャフトも一緒に回る

様々な応力と強度計算の基礎定義

  • 設計応力 σd ：設計者が「このぐらいの値に収めようとする目標値」
  • 許容応力 σal (= σF / S) ：設計者が定める，実用上の限界
  • 基準強さ σF ：部材の限界値（静的：引っ張り強さ等，動的：疲れ限度）
  • 作用応力 ：実際に部材に作用する応力（理想的には，設計応力と等しい）

・ キー幅：b = 16 [mm]
・ キー高さ：h = 10 [mm]
(シャフト側のキー溝深さ：t₂ = 6 [mm])
・ キー溝長さ：L = 60 [mm]
・ キー溝底隅角部の曲率半径：r = 0.4 [mm]

・ 幅比： b / D = 16 / 50 = 0.32
・ 深さ比： t₂ / D = 6 / 50 = 0.12
・ フィレット比： r / D = 0.4 / 50 = 0.008
結論： 規格表（キーみぞを有する軸の応力集中規格）において該当箇所を見ると，切り欠き係数 β は β = 3.3 〜 3.8 となっており，今回は安全側を考慮して β = 3.5 とした．
